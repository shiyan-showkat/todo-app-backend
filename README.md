# 🚀 Todo App Backend (MERN + Auth)

A scalable REST API for managing todos with **JWT Authentication, OTP Verification, and Secure User-based Data Handling**.

Built using **Node.js, Express, MongoDB (Mongoose)**.

---

## 🔥 Features

### 🧾 Todo Features

- ➕ Create Todo
- 📋 Get User Todos
- ✏️ Update Todo
- 🗑️ Delete Todo
- 🚫 Prevent duplicate todos (per user)

### 🔐 Authentication

- 📧 Signup with OTP verification
- ✅ Verify OTP
- 🔑 Login with JWT (Access + Refresh Token)
- 🔄 Auto refresh token system
- 🚪 Logout (cookie-based)

### 🔑 Password Recovery

- 📩 Forgot Password (OTP)
- ✅ Verify OTP
- 🔁 Reset Password

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (Authentication)
- bcrypt (Password hashing)
- cookie-parser
- dotenv
- cors
